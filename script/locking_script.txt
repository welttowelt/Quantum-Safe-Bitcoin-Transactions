//   This makes the scheme quantum-safe (hash-based only).

// (sig, sighash). This pins the entire transaction: any
// modification invalidates the sighash, breaking CHECKSIG.
// Quantum-safe because the sig itself is a constant;
// the binding comes from the hash, not from EC hardness.
//
